# Лендинг Юлии Ребровой

Статический адаптивный прототип сайта художницы-портретиста. Контент, цены и условия собраны из клиентского прайса.

Шрифты Prata, Old Standard TT и Golos Text хранятся локально в `assets/fonts/` и распространяются по лицензии SIL Open Font License (тексты лицензий лежат рядом с файлами шрифтов).

## Локальный запуск

```powershell
python -m http.server 4173
```

Откройте `http://localhost:4173`.

## SEO и аналитика

- OG-превью: `assets/og/yulia-rebrova-og.png` (1200×630).
- JSON-LD в `index.html`: `WebSite`, `ProfilePage`, `Person`, `ProfessionalService`, `Service` и каталог предложений.
- Событийная модель в `analytics.js`: CTA, заявка, сертификат, фильтры, карусели и глубина просмотра. Имена и текст заявки в аналитику не передаются.
- Чтобы включить внешние счётчики, укажите реальные значения в мета-тегах `ga4-measurement-id` и `yandex-metrika-id`. Провайдеры загружаются только после вызова `siteAnalytics.grantConsent()`.
- После выбора рабочего домена замените корневые значения `/` у `canonical`, `og:url` и URL в JSON-LD на абсолютный HTTPS-адрес. Тогда же добавьте абсолютный `sitemap.xml` в `robots.txt` и панели Google Search Console / Яндекс Вебмастер.

## Перед публикацией

1. Указать рабочий домен и идентификаторы аналитики.
2. Подключить выбранный интерфейс согласия на аналитику и политику конфиденциальности.
3. Дополнить оставшиеся карточки реальными работами и уточнить форматы/сроки у изображений с пометкой «Уточняется».
4. Проверить публичный URL в Rich Results Test, Schema.org Validator, Google Search Console и Яндекс Вебмастере.
