# Лендинг Юлии Ребровой

Статический адаптивный прототип сайта художницы-портретиста. Контент, цены и условия собраны из клиентского прайса.

Шрифты Prata, Old Standard TT и Golos Text хранятся локально в `assets/fonts/` и распространяются по лицензии SIL Open Font License (тексты лицензий лежат рядом с файлами шрифтов).

## Локальный запуск

```powershell
python -m http.server 4173
```

Откройте `http://localhost:4173`.

## Перед публикацией

1. Подключить реальные контакты художницы (Telegram / WhatsApp / телефон) в блоке `#contact`.
2. Заменить сюжетные SVG-иллюстрации реальными фотографиями работ, когда будет согласовано портфолио.
3. Подключить форму к выбранному каналу заявок или CRM. Сейчас она честно копирует подготовленный текст в буфер обмена.
4. Уточнить написание имени: в исходном запросе указано «Юрия», в прайсе — «Юлия Реброва». На сайте использовано имя из прайса как источник истины.
